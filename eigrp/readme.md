## EIGRP Quick Info

* Routing Protocol Type: Advanced Distance Vector
* Administrative Distance
    * Internal Routes: 90
    * External Routes: 170
* IP Protocol: 88
* Multicast address: 224.0.0.10
* Default Metric Calculation: Bandwidth and Delay
